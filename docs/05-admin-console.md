# 05 Admin Console

## 目标

定义内部运营台的页面结构、主要交互、关键字段和审核操作，作为前端实现依据。

## 页面清单

- 仪表盘
- 论文池
- 论文详情
- 文章编辑器
- 发布中心
- 配置中心
- 日志与任务中心

## 仪表盘

显示关键指标：

- 今日抓取数
- 过滤后数量
- 推荐数量
- 草稿生成数
- 发布成功数
- 失败任务数

## 论文池

核心能力：

- 列表筛选
- 查看评分
- 查看是否已生成草稿
- 手动触发重评分
- 排除论文

## 文章编辑器

核心能力：

- 编辑标题
- 编辑摘要
- 编辑正文 HTML
- 编辑标签
- 查看原论文信息
- 查看评分理由
- 预览公众号效果

## 发布中心

核心能力：

- 查看发布状态
- 失败重试
- 查询微信返回结果

## 配置中心

核心能力：

- 管理关键词
- 管理分类白名单
- 管理 Prompt 模板
- 管理自动发布开关

## 设计原则

- 以效率优先，不追求营销型 UI
- 所有高风险操作有确认
- 关键状态和失败原因必须可见

## 与代码生成的关系

本文件会直接影响：

- Next.js 路由
- 页面组件结构
- 表单字段设计
- API 调用封装

## 需要你补充的信息

- 是否必须支持富文本编辑
- 是否需要多人同时编辑锁
- 是否需要深色模式或移动端适配
- 是否需要接入组织内统一登录
