# 00 Overview

## 目标

定义系统的产品目标、V1 边界、核心产出和整体协作方式，使后续各模块实现都围绕同一套业务目标展开。

## 系统定位

本项目不是通用网页爬虫，而是一个面向 arXiv 遥感基础模型论文的内容生产工作流系统。

系统主链路：

1. 自动发现候选论文
2. 自动筛选最优论文
3. 自动解析与生成公众号草稿
4. 人工审核与编辑
5. 审核后发布到微信公众号

## V1 范围

V1 只做：

- 基于 arXiv API 的论文发现
- 每日推荐 1 篇论文
- 自动生成公众号草稿
- 人工审核后发布
- 基础日志、失败重试、后台可编辑

V1 不做：

- 多论文源聚合
- 自动无人审核直发
- 自动搬运论文图表并对外长期分发
- 多 Agent 自主上网浏览和自由决策

## 核心产出

- 候选论文池
- 今日推荐论文
- 结构化审稿内容
- 公众号 HTML 草稿
- 审核记录与发布记录

## 成功指标

- 每日抓取成功率高
- 推荐结果相关性稳定
- 草稿生成成功率高
- 人工审核耗时可控
- 发布链路可回溯
